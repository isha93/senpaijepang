# Appium Automation Docs

This folder contains Appium automation plan and usage for the Senpai Jepang mobile apps.

- `PLAN.md` — roadmap and scope
- `TEST-RUN-2026-03-05.md` — full Appium run record (iOS + Android)
- `SELECTORS.md` — canonical accessibility IDs (including register + verify email flow)
- Register+verify regression spec is gated by env `RUN_REGISTER_VERIFY=1` to avoid accidental account spam on default runs.
- Video output default path: `apps/qa-automation-appium/reports/videos`
- App code: `apps/qa-automation-appium`
